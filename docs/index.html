§ 01 / WHAT IT DOES

Every Claude Code session is full of decisions, commands, and context that vanish the moment you close the tab. journal-recorder captures all of it — automatically — so you can search, reference, and hand off your work months later.

W/01
What was built
Exact files created or modified, commands run, errors hit and how they were resolved.
W/02
Why decisions were made
Trade-offs, reasoning, and architectural choices — written for someone with zero prior context.
W/03
What's next
Action items extracted automatically. A new engineer could pick up exactly where you left off.
§ 02 / WHEN IT FIRES
◉

auto
◎

auto
◈

auto

§ 03 / INSTALL

Requirements: Claude Code · jq (brew install jq)

01 /

Install the agent

# Drop it into ~/.claude/agents/ — Claude Code picks it up automatically
mkdir -p ~/.claude/agents
curl -o ~/.claude/agents/journal-recorder.md \
  https://raw.githubusercontent.com/GauravRatnawat/journal-recorder-agent/main/journal-recorder.md
No restart needed. The agent is available immediately.
02 /

Enable auto-journaling on compaction (optional but recommended)

# Install the PostCompact hook
mkdir -p ~/.claude/hooks
curl -o ~/.claude/hooks/post-compact-journal.sh \
  https://raw.githubusercontent.com/GauravRatnawat/journal-recorder-agent/main/post-compact-journal.sh
chmod +x ~/.claude/hooks/post-compact-journal.sh

Add to ~/.claude/settings.json (merge into existing "hooks" if present):

{ "hooks": { "PostCompact": [ { "hooks": [ { "type": "command", "command": "bash ~/.claude/hooks/post-compact-journal.sh", "statusMessage": "Saving journal entry...", "timeout": 90 } ] } ] } }
§ 04 / WHAT ENTRIES LOOK LIKE

Entries are saved to ~/claude-journal/ as YYYY-MM-DD_HH-MM_<title>.md. Each one is self-contained — written for a new engineer with zero prior context.

~/claude-journal/2026-04-23_01-00_postcompact-hook-setup-and-fix.md
# PostCompact Hook Setup and Fix ## TL;DR Configured Claude Code to auto-journal on compaction. Initial PreCompact + agent hook failed (unsupported outside REPL). Switched to PostCompact + command hook that reads the conversation transcript and generates a full entry via claude -p. ## What Was Accomplished - Installed journal-recorder.md into ~/.claude/agents/ - Wrote post-compact-journal.sh with transcript extraction logic - Added PostCompact hook to ~/.claude/settings.json - Fixed permission issue on hook script (chmod +x) - Verified hook fires on /compact and auto-compact ## Key Decisions - **Decision**: PostCompact over PreCompact **Why**: PreCompact hooks are unsupported outside the REPL. PostCompact fires reliably and has access to the full transcript before it is pruned. ## Action Items - [ ] Test hook on next session in a different project - [ ] Consider adding a daily summary cron job ## Tags `#claude-code` `#hooks` `#automation` `#journal`
